## Getting Started

### Prerequisites

- Go 1.24+

### Build

```bash
make build
```

### Run

```bash
make run
```

### Clean

```bash
make clean
```

## Controls

| Key       | Action    |
|-----------|-----------|
| `q`       | Quit      |
| `Ctrl+C`  | Quit      |
